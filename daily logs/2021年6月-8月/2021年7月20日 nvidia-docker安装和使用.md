# 2 docker命令

```shell
docker run命令

# 参数
--name="Name" # 容器名字
-d            # 以后台方式启动
-it          # 交互方式运行，进入容器查看内容
-p			# 指定容器的端口
	-p 主机端口：容器端口（常用
	-p 容器端口
	容器端口
-P			# 随机指定端口

```
